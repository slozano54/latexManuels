\begin{exercice*}
    On donne les égalités : \\
    $415 = 7\times59+2$ \; et \; $56\times57 = \num{3 192}$. \\
    Sans effectuer de calculs, donner le quotient et le reste des divisions euclidiennes suivantes.
    \begin{multicols}{2}
    \begin{enumerate}
       \item 415 par 7
       \item \num{3 192} par 56
       \item 415 par 59
       \item \num{3 192} par 57
    \end{enumerate}
   \end{multicols}
   \hrefMathaleaVIII{37267}{eleve}{1211}
 \end{exercice*} 
 \begin{corrige}
    \phantom{rrr}\\\smallskip
    \begin{changemargin}{-5mm}{-5mm}
    \begin{enumerate}
       \item Le quotient de 415 par 7 est {\red 59}, le reste est {\red 2}.
       \item Le quotient de \num{3 192} par 56 est {\red 57}, le reste est {\red 0}.
       \item Le quotient de 415 par 59 est {\red 7}, le reste est {\red 2}.
       \item Le quotient de \num{3 192} par 57 est {\red 56}, le reste est {\red 0}.
    \end{enumerate}
   \end{changemargin}
 \end{corrige}
