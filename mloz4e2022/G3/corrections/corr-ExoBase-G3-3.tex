    Sur les calculatrices, une touche du type \Calculatrice{/$\sqrt{\phantom{rrr}}$/$x^2$}, permet de calculer un carré ou une racine carrée.
    \begin{enumerate}
        \item Repérer cette touche sur la calculatrice.
        \item Compléter ces tableaux à l'aide de la calculatrice.\\
        \medskip
        \begin{ctableau}{\linewidth}{9}
            \hline
            $a$&$\num{0.81}$&$\num{1.21}$&$\num{2.25}$&$\num{12.96}$&$\num{289}$\\\hline
            $\sqrt{a}$&&&&&\\\hline
        \end{ctableau}

        \medskip

    \end{enumerate}
