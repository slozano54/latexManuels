\begin{exercice*}
    Utiliser l'égalité des produits en croix pour déterminer le nombre manquant dans chacune des égalités suivantes.
    \begin{multicols}{2}
        \begin{enumerate}
            \begin{spacing}{2}
                \item $\dfrac{12}{56}=\dfrac{\dots{}}{\num{2.8}}$
                \item $\dfrac{-26}{65}=\dfrac{56}{\dots{}}$
                \item $-\dfrac{\num{-3.4}}{\num{-1.02}}=\dfrac{\num{-0.85}}{\dots{}}$
                \item $\dfrac{\dots{}}{\num{8.4}}=\dfrac{\num{-8.4}}{\num{-0.7}}$                 
            \end{spacing}
        \end{enumerate}           
    \end{multicols}
\end{exercice*}

    Utiliser l'égalité des produits en croix pour déterminer le nombre manquant dans chacune des égalités suivantes.

    \begin{enumerate}
        \begin{spacing}{2}
            \item $\dfrac{12}{56}=\dfrac{\dots{}}{\num{2.8}}$ {\red Les fractions sont égales donc les produits en croix sont égaux.

            $12\times\dots{}=56\times\num{2.8}$ ; on divise par $12$

            donc le nombre manquant vaut $\num{0.6}$.            
            }
            \item $\dfrac{-26}{65}=\dfrac{56}{\dots{}}$ {\red Les fractions sont égales donc les produits en croix sont égaux.

            $-26\times\dots{}=65\times 56$ ; on divise par $-26$

            donc le nombre manquant vaut $-140$.            
            }
        \end{spacing}
    \end{enumerate}
    \Coupe
    \begin{enumerate}
        \setcounter{enumi}{2}
        \begin{spacing}{2}
            \item $-\dfrac{\num{-3.4}}{\num{-1.02}}=\dfrac{\num{-0.85}}{\dots{}}$ {\red L'égalité revient à $\dfrac{\num{-3.4}}{\num{1.02}}=\dfrac{\num{-0.85}}{\dots{}}$
            
            Les fractions sont égales donc les produits en croix sont égaux.

            $\num{-3.4}\times \dots{}=\num{1.02}\times \num{-0.85}$ ; on divise par $\num{-3.4}$

            donc le nombre manquant vaut $\num{0.255}$.            
            }
            \item $\dfrac{\dots{}}{\num{8.4}}=\dfrac{\num{-8.4}}{\num{-0.7}}$ {\red Les fractions sont égales donc les produits en croix sont égaux.

            $\dots{}\times\num{-0.7}=\num{8.4}\times\num{-8.4}$ ; on divise par $\num{-0.7}$

            donc le nombre manquant vaut $\num{100.8}$.            
            }
        \end{spacing}
    \end{enumerate}           
\end{corrige}
