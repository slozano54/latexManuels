\begin{exercice}
    Parmi les nombres suivants, lequel est un nombre entier négatif ?    
    \begin{ChoixQCM}{4}
        \item $+5$
        \item $9$
        \item $-7$
        \item $-4,3$
    \end{ChoixQCM}
\end{exercice}
\begin{corrige}
    Réponse \reponseQCM{c}, $-7$ est un nombre entier négatif, car :
    \begin{itemize}
        \item Sa distance à zéro est un nombre entier.
        \item Elle est précédée par le signe $-$.
    \end{itemize}
\end{corrige}
